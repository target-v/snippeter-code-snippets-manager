Installation

To properly install Snippeter follow these steps:

- Extract Snippeter archive

- Create database for Snippeter

- Import all tables in database.sql which you can find in 'Database' folder

- Upload all files to server with 'read' permissions

- Set 'write' permission to 'Tmp' folder

 

After you completed previous steps now you need to tweak some files so script can connect to your server and database.

Open 'connect.php' inside 'Database' folder. File looks like this:

<?php

    $host = 'localhost';
    $username = 'root';
    $password = '';
    $database = 'snippeter';

    $pageRoot = 'http://example.com'; // without " / " at the end
    $yourEmail = 'yourEmail@gmail.com';
    $numUsers = 20; // number of users per page in admin panel
    $numSnip = 20; // number of snippets per page in admin panel

    $con = new mysqli($host, $username, $password, $database);

    if (mysqli_connect_errno()) {
        printf("Connect failed: %s\n", "Server error");
        exit();
    }
    
?>

Enter data from your server in corresponding fields, host, username, password and database.

'$pageRoot' is web address where you uploaded Snippeter. So for example if your domain is www.example.com and you uploaded the script in folder called 'snippeter' then your '$pageRoot' will look like this:

$pageRoot = 'http://example.com/snippeter'

'$yourEmail' is email which users see when they register to Snippeter, make sure it is valid email.

Now all installation steps are completed.

Usage

These are some basic tips in order to get you up and running with Snippeter.

- So after you successfully installed script open index.php and you will be presented with main page where you can register or login.

- Registration needs only basic information afterwards you need to verify your account by following instructions in email which you received after successful registration.

- After you logged in, you'll see blank template. To add new snippet click on " + "sign in bottom left corner. Fill in needed information like name, description, snippet, tags and group.

- Snippets can be categorized into groups. To create a group click on folder sign in bottom left corner, give it a name and click Save.

- You can remove the snippet by hovering with your mouse over snippet name and by pressing od "trash" icon

- Snippets can also be categorized with tags

- If you made a mistake, you can edit your snippet by hovering with your mouse over it and selecting "edit" icon.

- Sublime Text is one of the most popular text editor for programmers, so I decided to make Export option for Sublime Text. Now you can have all your snippets in Snippeter exported for Sublime Text. When you export all snippets at once you can't assign tabTrigger for snippets, but if you decide to download one at the time then you can assign tabTrigger. TabTrigger is word which brings your snippet to text editor after you type it and press Tab key.

- Snippets can be also exported in JSON file and plain text.

- Importing snippets is simple as clicking on Import option and selecting valid JSON file which contains snippets.

- All snippets can be Public and Private. Public snippet get their own link which you can share with others.

- For quick snippet finding press Ctrl + F on your keyboard and you'll bring the search bar in focus, after you found your snippet press ENTER and snippet on top will be opened.

- Settings dialog you can change Font, Font size, disable or enable Line numbers, email and password.

- Also you can use Copy option to send current snippet to clipboard or Code option to get raw code in new tab.

Admin Panel

Admin panel contains:

- Dashboard

- Snippets

- Users list

- Banned users list

- Settings

To access Admin panel run admin-login.php

 

Dashboard gives you basic information about users and snippets. You can see how many users registered to your site, how many of them are online, how many snippets are online etc.

Snippets gives you full controll over all snippets on your server. You can preview, edit and delete every snippet.

Users list gives you control over your users. You can manually Add new user, delete it or ban it. Admins can also use search bar to quickly find specific user.

Banned users list gives you a list of users who are banned by admin. Those users can't login until you unban them.

Settings section gives you abillity to change your password and email.

Updating

Updates will be released to fix bugs and/or to bring new features to Snippeter. To successfully update your existing installation follow these steps:

- Download and extract new archive from CodeCanyon

- Check for changes in database/connect.php, if there is no changes then do not copy this file to your existing installation

- Copy all files to your server

- Test installation
